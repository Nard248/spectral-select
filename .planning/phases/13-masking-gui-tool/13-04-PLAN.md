<task type="auto">
  <name>Task 1: Add masking toolbar and drawing tools</name>
  <files>spectral_select/viewer.py</files>
  <action>
Implement mask drawing tools:

1. **Masking Toolbar** (in Tools section of Control Panel):
   - Tool buttons (icon-style or text):
     - [Pointer] - Default, no drawing (selection/pan mode)
     - [Polygon] - Polygon drawing mode
     - [Rectangle] - Rectangle drawing mode
     - [Brush] - Freehand brush (stretch goal)
   - Active tool highlighted
   - Cursor changes based on tool

2. **Polygon Drawing**:
   - Click to add vertices
   - Line preview connects points
   - Double-click or Enter to close polygon
   - Right-click or Escape to cancel
   - Minimum 3 points required
   - Visual feedback: red outline while drawing, filled preview on close

3. **Rectangle Drawing**:
   - Click for first corner
   - Drag to opposite corner (rubber-band preview)
   - Release to complete
   - Or: Click two points for corners
   - Visual: dashed rectangle during draw

4. **Drawing state management**:
   - self.current_tool: str ('pointer', 'polygon', 'rectangle', 'brush')
   - self.drawing_points: list of (x, y) tuples
   - self.drawing_active: bool
   - self.preview_patch: matplotlib patch for preview

5. **Mask creation from drawing**:
   - Use matplotlib.path.Path.contains_points() for polygon
   - Simple numpy slicing for rectangle
   - Convert to binary mask (height, width) dtype=uint8

6. **Helper functions** (module-level):
   ```python
   def create_polygon_mask(points: list[tuple[float, float]], height: int, width: int) -> np.ndarray:
       """Create binary mask from polygon vertices."""

   def create_rectangle_mask(corner1: tuple[int, int], corner2: tuple[int, int], height: int, width: int) -> np.ndarray:
       """Create binary mask from rectangle corners."""
   ```
  </action>
