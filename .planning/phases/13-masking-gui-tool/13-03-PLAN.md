  <action>
Add spectral profile visualization:

1. **Spectrum Panel** (below or beside main canvas, collapsible):
   - matplotlib Figure (400x200 default) showing spectrum plot
   - X-axis: Emission wavelength (nm) or band index
   - Y-axis: Intensity value
   - Title: "Spectrum at (x, y)" updated on click

2. **Click interaction**:
   - Left-click on main canvas: Extract spectrum at that pixel
   - Get all band values: spectrum = cube[:, y, x]
   - Plot as line with markers
   - Show crosshair or marker on main image at clicked location

3. **Multi-point comparison**:
   - Checkbox: "Compare mode"
   - When enabled, clicks add spectra to plot (different colors)
   - Legend shows pixel coordinates for each trace
   - "Clear" button removes all traces
   - Maximum 5 traces (cycle colors)

4. **Spectrum plot features**:
   - Grid lines for readability
   - Axis labels with units
   - Auto-scale Y-axis or fixed range toggle
   - Export spectrum data button (CSV with wavelength, value columns)

5. **Multi-excitation spectrum** (advanced):
   - Option to show spectrum across ALL excitations for clicked pixel
   - Separate subplot or combined with different line styles
   - Useful for seeing full 4D response at a location

6. **Helper function** (module-level):
   ```python
   def extract_spectrum(cube: np.ndarray, x: int, y: int) -> np.ndarray:
       """Extract spectrum at pixel (x, y) from cube."""

   def extract_multi_excitation_spectrum(spectra_data: SpectraData, x: int, y: int) -> dict[float, np.ndarray]:
       """Extract spectra at pixel across all excitations."""
   ```
