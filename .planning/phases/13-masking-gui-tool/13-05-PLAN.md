1. **Main function with argparse**:
   ```python
   def main():
       parser = argparse.ArgumentParser(
           description='ME-HSI Viewer - Multi-Excitation Hyperspectral Image Viewer',
           formatter_class=argparse.RawDescriptionHelpFormatter,
           epilog='''
   Examples:
     python -m spectral_select.viewer                    # Open empty viewer
     python -m spectral_select.viewer data.pkl          # Open with data file
     python -m spectral_select.viewer --mask mask.npy   # Open with mask
           '''
       )
       parser.add_argument('file', nargs='?', help='Data file to open (.pkl or .im3)')
       parser.add_argument('--mask', '-m', help='Mask file to load (.npy)')
       parser.add_argument('--excitation', '-e', type=float, help='Initial excitation wavelength')
       parser.add_argument('--geometry', '-g', help='Window geometry WxH+X+Y (e.g., 1200x800+100+100)')
       args = parser.parse_args()
   ```
