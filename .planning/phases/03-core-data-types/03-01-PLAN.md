<task type="auto">
  <name>Task 3: Create SpectraData dataclass</name>
  <files>spectral_select/types.py</files>
  <action>
Create a SpectraData dataclass as the main container for multi-excitation hyperspectral data:
- excitations: Dict[float, ExcitationData] (mapping excitation wavelength to data)
- mask: Optional[np.ndarray] = None (binary mask, 1=valid, 0=masked)
- sample_name: str = "sample" (identifier for the sample)
- loading_options: Optional[LoadingOptions] = None (preprocessing settings used)
- metadata: Dict[str, Any] = field(default_factory=dict) (arbitrary metadata)

Properties:
- excitation_wavelengths: List[float] (sorted list of excitation wavelengths)
- n_excitations: int (number of excitation wavelengths)
- spatial_shape: Tuple[int, int] (height, width from first excitation)

Methods:
- get_excitation(ex_nm: float) -> ExcitationData (get data for specific excitation)
- to_dict() -> Dict[str, Any] (serialize for JSON, excluding large arrays)

Class methods for loading:
- from_pickle(path: Union[str, Path]) -> "SpectraData"
  Load from processed pickle file format (the existing pkl format with 'data' and 'excitation_wavelengths' keys)
- from_dict(data: Dict[str, Any]) -> "SpectraData"
  Create from dictionary (for deserialization)

Do NOT implement from_im3() in this phase - that requires ImageJ/external dependencies and belongs in Phase 4 (Analysis Engine) or a separate data loading module.

Add __post_init__ validation:
- All excitations must have same spatial dimensions
- Mask shape must match spatial dimensions if provided
  </action>
