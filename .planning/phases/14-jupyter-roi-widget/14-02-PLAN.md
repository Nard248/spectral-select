  <name>Task 2: Implement to_ground_truth() export and SpectraData factory</name>
  <files>spectral_select/widgets.py</files>
  <action>
    1. Add to_ground_truth() method to ROIWidget:
       ```python
       def to_ground_truth(self) -> "GroundTruth":
           """Export ROI labels as GroundTruth for validation.

           Returns:
               GroundTruth instance with labels array where:
               - -1 = background (no ROI selected)
               - 0, 1, 2... = class labels

           Raises:
               ValueError: If no ROIs have been drawn.
           """
       ```
       - Build labels array from _class_labels (start at 0, -1 for background)
       - Build color_mapping from _class_colors (convert color names to RGBA tuples)
       - Include class_names from _class_names
       - Import GroundTruth inside method to avoid circular imports

    2. Add class method from_spectra_data():
       ```python
       @classmethod
       def from_spectra_data(
           cls,
           data: "SpectraData",
           excitation: Optional[float] = None,
           figsize: Tuple[float, float] = (8, 6),
       ) -> "ROIWidget":
           """Create ROI widget from SpectraData.

           Convenience factory that handles excitation selection.
           """
       ```
       - Validate excitation is in data.excitation_wavelengths
       - Return cls(data, excitation, figsize)

    3. Add save_mask(path: Path) and load_mask(path: Path) methods:
       - Save: Export combined mask as PNG (class_id encoded as pixel value)
       - Load: Import PNG and populate _class_labels from unique values
       - Use PIL.Image for I/O (already a dependency)
  </action>
