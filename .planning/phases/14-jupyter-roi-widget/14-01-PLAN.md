<task type="auto">
  <name>Task 2: Implement ROIWidget class with LassoSelector</name>
  <files>spectral_select/widgets.py</files>
  <action>
    Create ROIWidget class in widgets.py:

    ```python
    class ROIWidget:
        """Interactive ROI selection widget for Jupyter notebooks.

        Displays a hyperspectral image and allows drawing ROIs using
        matplotlib's LassoSelector. Requires %matplotlib widget magic.

        Example:
            %matplotlib widget
            from spectral_select import SpectraData, ROIWidget

            data = SpectraData.from_pickle("sample.pkl")
            widget = ROIWidget(data)
            widget.display()

            # After drawing ROI
            mask = widget.get_mask()
        """
    ```

    Implementation:
    - __init__(self, data: SpectraData, excitation: Optional[float] = None, figsize=(8, 6)):
      - Store data reference, select first excitation if not provided
      - Initialize private attributes: _mask (None), _selector (None), _fig, _ax
      - Create ipywidgets.Output() for capturing matplotlib output

    - display(self) -> None:
      - Use output widget context manager to capture figure
      - Create figure/axes with figsize
      - Display image using create_display_image helper
      - Initialize LassoSelector with _on_select callback
      - Add instruction text below figure using ipywidgets.Label
      - Return VBox([output, label])

    - _on_select(self, vertices) -> None:
      - Convert vertices to mask using path_to_mask helper
      - Store in self._mask
      - Overlay semi-transparent mask on image (alpha=0.3, color='red')
      - Call fig.canvas.draw_idle() to update

    - get_mask(self) -> Optional[np.ndarray]:
      - Return copy of self._mask or None if no selection

    - clear(self) -> None:
      - Reset self._mask to None
      - Redraw image without overlay

    Use properties for read-only access to fig, ax, mask. Include type hints throughout.
  </action>
