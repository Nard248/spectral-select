  <name>Task 1: Create ResultsManager class with directory structure</name>
  <files>spectral_select/results.py</files>
  <action>
Create new module `spectral_select/results.py` with ResultsManager dataclass:

```python
@dataclass
class ResultsManager:
    """Manages structured output organization for analysis runs."""

    base_dir: Path  # e.g., results/
    sample_name: str
    run_id: Optional[str] = None  # Auto-generated if None

    # Computed paths (set in __post_init__)
    run_dir: Path = field(init=False)
    model_dir: Path = field(init=False)
    viz_dir: Path = field(init=False)
    layers_dir: Path = field(init=False)
```

Implement:
1. `__post_init__` - Generate run_id if None (format: YYYYMMDD_HHMMSS), create directory paths, call `ensure_directories()`
2. `ensure_directories()` - Create all subdirectories (run_dir, model_dir, viz_dir, layers_dir)
3. `get_model_path(checkpoint_type: str = "final") -> Path` - Return path for model checkpoint (best/final)
4. `get_result_path(filename: str) -> Path` - Return path for result file in run_dir
5. `get_layer_path(layer_index: int) -> Path` - Return path for TIFF layer
6. `update_latest_symlink()` - Create/update `latest` symlink pointing to current run_dir
7. Class method `from_config(config: Config, base_dir: Optional[Path] = None)` - Factory from Config

Use pathlib throughout. Validate sample_name is not empty. Run ID format: 8-digit date + 6-digit time (e.g., 20260120_143052).

Do NOT modify any other files yet - this is the foundation module.
