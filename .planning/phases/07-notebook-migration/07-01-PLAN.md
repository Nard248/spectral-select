**spectral_select public API:**
- `Analyzer` - main analysis class with fit(), get_wavelengths()
- `Config` - configuration dataclass
- `Visualizer` - visualization with from_result(), from_analyzer() factories
- `Validator` - ground truth validation with fit(), score()
- `SpectraData` - data container with from_pickle() loader
- `WavelengthResult`, `WavelengthBand` - result types

**Example workflow:**
```python
from spectral_select import Analyzer, Config, SpectraData, Visualizer

# Load data
data = SpectraData.from_pickle("processed_data.pkl")

# Configure and run analysis
config = Config(sample_name="Lichens_2", n_bands_to_select=30)
analyzer = Analyzer(config)
analyzer.fit(data)

# Get results
bands = analyzer.get_wavelengths()

# Visualize
viz = Visualizer.from_analyzer(analyzer)
viz.plot_influence_heatmap()
viz.plot_wavelength_scatter()
```
