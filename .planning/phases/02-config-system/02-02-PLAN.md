  <files>spectral_select/config.py</files>
  <action>
Add class methods for loading configurations:

**from_dict(cls, data: Dict[str, Any]) -> "Config":**
- Takes a dictionary and creates a Config instance
- Handle Path conversion: if field is Optional[Path] and value is str, convert to Path
- Handle component fields: if value is dict with "type" key, use that for resolution
- Skip unknown keys with a warning (for forward compatibility)
- Use dataclasses.fields() to iterate over expected fields

**from_yaml(cls, path: Union[str, Path]) -> "Config":**
- Read YAML file using yaml.safe_load (import yaml at top)
- Call from_dict with loaded data
- Raise FileNotFoundError with clear message if file doesn't exist
- Raise ValueError if YAML is malformed

**from_json(cls, path: Union[str, Path]) -> "Config":**
- Read JSON file using json.load
- Call from_dict with loaded data
- Same error handling as from_yaml

Add import at top: `import yaml` (PyYAML is already in requirements.txt)

IMPORTANT: Use try/except for yaml import and raise helpful error if not installed.
  </action>

<task type="auto">
  <name>Task 2: Add to_yaml and to_dict instance methods</name>
  <files>spectral_select/config.py</files>
  <action>
Add instance methods for serialization:

**to_dict(self) -> Dict[str, Any]:**
- Convert Config to dictionary suitable for JSON/YAML serialization
- Convert Path objects to strings
- Handle pluggable components:
  - If string identifier, keep as string
  - If class/callable, store {"type": "custom", "class": qualified_name}
- Use dataclasses.asdict() as base, then transform

**to_yaml(self, path: Union[str, Path]) -> None:**
- Convert to dict using to_dict()
- Write to file using yaml.safe_dump with default_flow_style=False for readability
- Create parent directories if needed (path.parent.mkdir(parents=True, exist_ok=True))

**to_json(self, path: Union[str, Path]) -> None:**
- Convert to dict using to_dict()
- Write to file using json.dump with indent=2 for readability
- Create parent directories if needed

Add __repr__ method that shows key configuration values in a readable format.

Add __eq__ method for comparing configurations (useful for testing).
  </action>
