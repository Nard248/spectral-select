<task type="auto">
  <name>Task 2: Create GitHub Actions workflow for CI</name>
  <files>.github/workflows/test.yml</files>
  <action>
Create .github/workflows/test.yml with CI pipeline:

```yaml
name: Tests

on:
  push:
    branches: [main, main-cleanup]
  pull_request:
    branches: [main]

jobs:
  test:
    runs-on: ubuntu-latest
    strategy:
      matrix:
        python-version: ["3.11", "3.12"]

    steps:
      - uses: actions/checkout@v4

      - name: Set up Python ${{ matrix.python-version }}
        uses: actions/setup-python@v5
        with:
          python-version: ${{ matrix.python-version }}

      - name: Install dependencies
        run: |
          python -m pip install --upgrade pip
          pip install -e ".[dev]"

      - name: Run tests with coverage
        run: pytest --cov=spectral_select --cov-report=xml

      - name: Upload coverage to Codecov
        uses: codecov/codecov-action@v4
        with:
          file: ./coverage.xml
          fail_ci_if_error: false
```

Create .github/workflows/ directory first. Use actions/checkout@v4 and actions/setup-python@v5.
Include both Python 3.11 and 3.12 in matrix.
Codecov upload is optional (fail_ci_if_error: false).
