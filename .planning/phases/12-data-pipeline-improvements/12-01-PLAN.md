Add a from_raw() class method to SpectraData that uses DataLoader internally.

Signature:
```python
@classmethod
def from_raw(
    cls,
    data_path: Union[str, Path],
    metadata_path: Optional[Union[str, Path]] = None,
    mask: Optional[np.ndarray] = None,
    sample_name: Optional[str] = None,
    loading_options: Optional[LoadingOptions] = None,
    **loader_kwargs
) -> "SpectraData":
```

Implementation:
1. Create DataLoader with data_path and metadata_path
2. Call loader.load() to get raw data dict
3. Build ExcitationData objects from loaded cubes
4. Use provided mask or None if not provided
5. Derive sample_name from directory name if not provided
6. Return new SpectraData instance

Error handling:
- Re-raise DataLoadingError with context about the data path
- Validate that at least one excitation was loaded
- Log warning if metadata_path provided but couldn't be read

Do NOT:
- Change existing from_pickle behavior
- Add ImageJ as a hard import at module level
